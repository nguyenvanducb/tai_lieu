| **Bare Metal Solution**          | Máy chủ vật lý thật – dùng cho workload đặc biệt.                                |

1. Google Cloud Platform (GCP)
GCP là nền tảng điện toán đám mây của Google.
Nó cung cấp toàn bộ dịch vụ cloud, bao gồm:
Compute: máy ảo (GCE), Kubernetes (GKE), serverless (Cloud Run, Cloud Functions)
Storage: Cloud Storage, Persistent Disk, Filestore
Database: Cloud SQL, Firestore, BigQuery
Networking: VPC, Load Balancer, Cloud CDN
AI/ML, Security, DevOps Tools, …
Nói ngắn gọn: GCP là toàn bộ hệ sinh thái cloud, không chỉ là máy ảo.
2. Google Compute Engine (GCE)
GCE là dịch vụ cung cấp máy ảo (VM) trên GCP.
Cho phép bạn tạo và quản lý VM với nhiều OS, CPU/RAM, storage khác nhau.
Tương đương với EC2 trên AWS.
Ví dụ minh họa:
Bạn muốn chạy app trên máy ảo, bạn dùng GCE.
Bạn muốn chạy container: bạn dùng GKE.
Bạn muốn serverless: bạn dùng Cloud Run.
✅ Tóm lại:
GCP = nền tảng cloud đầy đủ
GCE = dịch vụ máy ảo (VM) trong GCP
Nếu bạn muốn, mình có thể vẽ sơ đồ các dịch vụ compute chính trên GCP để bạn thấy sự khác nhau giữa GCE, GKE, Cloud Run, Cloud Functions. Bạn có muốn mình làm sơ đồ không?
